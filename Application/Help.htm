AgOpenGPS Help

The files…

All the files to run are in the Application Directory. Run AgOpenGPS.exe to start the program.

Fields are stored in the MyDocuments folder and settings are stored in AppData\AgopenGPS so the application can be run from anywhere.

Connecting…

Connect a USB type GPS antenna to your computer or tablet that outputs an NMEA string. Take note of which port it connects to. It must output RMC and will work with only that.

If you want to see elevation, and more GPS information GGA is required also, but not absolutely required.

Running the App

 

**** You MUST CLICK ON SAVE to save your changes. If not, it won’t save. *******

 

Double click the file AgOpenGPS.exe to run the application.

First time it runs it knows nothing about its surroundings so it will say COM99 doesn’t exist and that you need to go into settings. Click OK.

It should pop up and show a grid and single section with vehicle.

Click on Settings -> COM Ports to start setting.

Click Port and choose the right one.

IF you don’t see your Port, click Rescan. If you still don’t, it’s a problem.

Click Baud and choose whatever your antenna prefers.

Click Connect.

**** You MUST CLICK ON SAVE to save your changes. If not, it won’t save.

You should see a stream of letters and numbers in the white text box that previously said nothing. Nothing will work till it shows an RMC stream of lines.

Settings -> Vehicle

 

Fore/Aft

 This sets how far the offset is of your antenna from the actual section. A positive value means its ahead of the antenna. A negative value means it behind.

It will show you in meters and feet.

Look Ahead

This value in seconds tells how early you want the section to turn on before you get to the line that already has been applied, like a headland.

It is speed compensated so the faster you go, the farther away it will turn on.

Antenna Height

How high up is the antenna.

**** You MUST CLICK ON SAVE to save your changes. If not, it won’t save.

Settings -> Sections

Here you define how many sections, and where along the tool each section begins and ends.

Positions on the left side are negative; positions on the right are positive – Very important!!

First click on the Sections spinner to set how many sections. You will notice the green highlighted sections match your setting.

                The little “tractor” in red moves along to help you visualize section placement.

Each spinner sets the start and endpoint of the section. Set accordingly, and make sure there are no negative boom widths.

Example:

                You have 3 booms each 4 meters wide. That’s a 12 meter boom.

                Set the section spinner to 3.

                Set the leftmost spinner to -6.0

                Set the rightmost spinner to 6.0

                The total width should be 12.0 meters or 39’ 4”

                The center of the boom is the center of the middle section so each extent of the boom is 2 over from center.

                Set the left center spinner to -2.0

                Set the right center spinner to 2.0

                Notice you have 3 booms, each 157” wide. If any boom widths are negative, you got a problem.

                There is no error checking here, but it will be obvious when you go to the main screen.

Settings -> Guidance
